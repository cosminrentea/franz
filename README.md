# franz

Install (after installing golang):
- go get github.com/astaxie/beego
- go get github.com/beego/bee
- gulp

Regenerate frontend:
- for development:
```
gulp
```
- for production:
```
gulp --env=prod 
```

Start server with:
```
bee run
```

Server will be available at:
```
http://localhost:8080
```
